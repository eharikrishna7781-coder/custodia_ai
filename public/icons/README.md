# Icons

Add your app icons here with the following naming convention:

| File | Size | Purpose |
|------|------|---------|
| `icon-72x72.png` | 72x72 | Small icon |
| `icon-96x96.png` | 96x96 | Favicon |
| `icon-128x128.png` | 128x128 | Chrome Web Store |
| `icon-144x144.png` | 144x144 | MS tile |
| `icon-152x152.png` | 152x152 | iPad touch icon |
| `icon-192x192.png` | 192x192 | Android icon |
| `icon-384x384.png` | 384x384 | Splash screen |
| `icon-512x512.png` | 512x512 | PWA icon |
| `apple-touch-icon.png` | 180x180 | Apple touch icon |
| `favicon-32x32.png` | 32x32 | Browser favicon |
| `favicon-16x16.png` | 16x16 | Browser favicon |
| `safari-pinned-tab.svg` | SVG | Safari pinned tab |

You can generate these icons using:
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

The icons should use the primary color #0D9488 (teal) for consistency with the app theme.
